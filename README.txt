
Urban Feed - Paquete listo para GitHub Pages
-------------------------------------------

Instrucciones rápidas:
1) Descomprime este zip.
2) Reemplaza las imágenes en /images por tus fotos reales (nombres deben coincidir con data/productos.json).
   - logo: images/logo-urbanfeed.png
   - menú fondo: images/menu-bg.jpg
   - sobre nosotros (imagen opcional): images/sobre-nosotros.jpg
3) Si querés usar otra ruta para productos, edita js/main.js -> const PRODUCTS_PATH.
4) Sube todo a tu repo y activa GitHub Pages (main / root).
5) Sube un archivo music/musica.mp3 si querés que suene al presionar el botón de música.
6) Para editar el catálogo desde Excel: abre UrbanFeed_Productos.xlsx, modifica y guarda.
   Para convertir a JSON, puedes usar un servicio online o pedir que te lo convierta.
